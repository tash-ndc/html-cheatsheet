Basic HTML Structure Tags

Tag Name Tag Syntax Description
DOCTYPE <!DOCTYPE html> First line of an HTML document. Used to inform the broswer that HTML5 is being used.
HTML <html></html> Second line of an HTML document. Used to inform the broswer that everything between these tags is HTML.
Head <head></head> Third line of an HTML document. Includes information that the user will not see.
Body <body></body> Contains the content of the page.
Header <header></header> Contains the main heading of the page.
Main <main></main> Contains the bulk content of the page.
Footer <footer></footer> Refers to the bottom of the page, where links to information such as FAQs and T&Cs can be found.
